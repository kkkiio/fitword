需求文档：词感 v0.1

一句话定位

一个本地运行的 LLM 表达训练工具，通过“选择题练词”和“写作评分”帮助用户把被动词汇转化为可调用表达。

目标用户

经常觉得自己“知道意思，但找不到合适词”的人。
想提升聊天、写作、评论、观点表达准确度的人。
希望围绕自己感兴趣的话题练习词汇和表达的人。
核心问题

传统表达训练门槛高，用户一开始很难凭空写出精准词。LLM 可以提供候选词、语境、解释和反馈，把训练从自由输出降到“辨认、选择、替换、改写”。

MVP 功能

选择题练习
用户输入一个话题，例如：

AI 新闻、咖啡店、职场沟通、电影评论、朋友聊天、产品体验

LLM 生成一道选词题：

一段短语境
4 个候选词
用户选择答案
LLM 给出正确答案和选项辨析
记录错题和用户不熟悉的词
题型示例：

名词辨析：某个东西、结构、现象叫什么
动词辨析：某个动作怎样描述更准
形容词辨析：性质、质感、风格如何表达
逻辑词辨析：因果、条件、让步、递进如何连接
领域词辨析：某个话题下的专业表达
写作评分
用户输入一段文字，LLM 返回：

总分：1-5
主要问题：用词粗、概念模糊、逻辑松、语气不自然等
可替换词建议
改写版本
根据这段文字生成 1-3 道针对性选择题
评分维度：

准确度：词是否贴合语境
具体度：是否过于笼统
自然度：是否像真实口语或自然书面语
结构：表达是否清楚
语域：是否符合聊天、评论、正式写作等场景
话题定制
用户可以输入：

一个关键词
一段新闻/文章/聊天内容
一个自己感兴趣的事件描述
LLM 根据输入生成练习素材。v0.1 优先支持用户粘贴文本；URL 抓取可放到后续版本。

本地使用形态
初版类似 roundtable：轻量、本地、快速启动。

建议形态：

text


1. 开始选择题练习
2. 写作评分
3. 输入自定义话题
4. 查看错题/收藏词
5. 退出
技术上可以先做成 CLI 或本地 Web UI。无需账号系统，无需数据库，数据存 JSON 文件即可。

成功标准

用户可以在 1 分钟内开始第一道练习。
用户不用自己先想出好词，也能完成训练。
每次练习后，用户至少获得 1 个可复用词语或表达。
写作评分能给出具体可改的建议，而不是泛泛评价。
