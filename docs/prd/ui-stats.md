# 统计 UI

练习数据可视化，展示本地练习记录的汇总结果。只读展示，v0.1 不做日期筛选和导出。

```
┌──────────────────────────────────────────────┐
│  练习统计                                    │
│                                              │
│  ┌──────┐ ┌──────┐ ┌──────┐                 │
│  │总题数│ │可用率│ │优质率│                  │
│  │ 35   │ │82.9% │ │62.9% │                  │
│  └──────┘ └──────┘ └──────┘                 │
│  待打磨率 37.1%                              │
│                                              │
│  薄弱类型分布                                 │
│  形容词  可用70% 优质40% 待打磨60%             │
│  名词    可用85% 优质75% 待打磨25%             │
│  动词    可用90% 优质85% 待打磨15%             │
│  逻辑词  可用80% 优质70% 待打磨30%             │
│  领域词  可用88% 优质80% 待打磨20%             │
│                                              │
│  选择 vs 填空                                 │
│  选择题  可用80% 优质80% 待打磨20%             │
│  填空题  可用72% 优质55% 待打磨45%             │
└──────────────────────────────────────────────┘
```

- 顶部：总题数、可用率、优质率、待打磨率概览
- 中部：按知识类别展示可用率、优质率和待打磨率，薄弱项视觉突出
- 下部：选择题 vs 填空题质量对比
- 待打磨率表示未达到好答案的比例
