# 统计 UI

统计页是本地练习记录的只读轻量概览。当前统计不是核心学习路径，只展示足够用户确认练习积累的摘要，不做复杂分析。

```
┌──────────────────────────────────────────────┐
│  练习统计                                    │
│                                              │
│  答题概览                                    │
│  总题数 35   可用率 82.9%   优质率 62.9%      │
│  待打磨率 37.1%                              │
│                                              │
│  选择题  20  优质率 80%                       │
│  填空题  15  优质率 55%                       │
│                                              │
│  写作评分概览                                │
│  评分次数 4   平均总分 3.5                    │
│  准确度 3.8  具体度 3.0  自然度 3.6            │
│  结构 3.4    语域 3.7                         │
└──────────────────────────────────────────────┘
```

- 答题概览展示总题数、可用率、优质率和待打磨率
- 选择题 / 填空题只展示题数和优质率，不展开复杂分布
- 写作评分概览展示评分次数、平均总分和五维平均分
- 没有数据时展示空态，不展示占位数字
- 不做日期筛选、导出、趋势图或手动编辑
