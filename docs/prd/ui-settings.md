# 设置 UI

通过侧边栏底部的齿轮图标打开设置弹窗。

## 入口

侧边栏底部，在「统计」下方显示齿轮图标按钮：

```
┌─────────────────┐
│  📊 统计        │
├─────────────────┤
│  ⚙              │  ← 齿轮图标，点击打开设置弹窗
└─────────────────┘
```

## 设置弹窗

居中 Dialog，点击齿轮图标触发。

```
┌──────────────────────────────────────────────┐
│                                              │
│       ┌───────────────────────┐              │
│       │  设置               ✕ │              │
│       ├───────────────────────┤              │
│       │                       │              │
│       │  界面语言              │              │
│       │  ○ 中文  ● English     │              │
│       │                       │              │
│       └───────────────────────┘              │
│                                              │
└──────────────────────────────────────────────┘
```

- 标题行：左侧「设置」，右侧关闭按钮 ✕
- 内容区：语言切换，两个按钮（中文 / English），当前选中高亮
- 点击弹窗外部或关闭按钮关闭
- 语言切换后弹窗不自动关闭，用户可继续修改后手动关闭
- 语言切换后，设置弹窗内的标题、提示和选项文案跟随当前界面语言更新
