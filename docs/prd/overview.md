# Product Requirements: Overview

## 背景

很多人在表达时遇到一种困境：明明认识某个词，也知道它是什么意思，但在需要表达时就是想不起来、用不出来。这种"被动词汇"远大于"主动词汇"的落差，在聊天、写作、评论、观点表达等场景中反复出现。

传统的表达练习门槛很高——用户一开始很难凭空写出精准的词。LLM 改变了这个局面：它可以提供候选词、语境、解释和反馈，把练习从"自由输出"降到"辨认、选择、替换、改写"，让用户不需要自己先想出好词，也能完成有效练习。

### 目标用户

1. 经常觉得自己"知道意思，但找不到合适词"的人。
2. 想提升聊天、写作、评论、观点表达准确度的人。
3. 希望围绕自己感兴趣的话题练习词汇和表达的人。

## 目标

1. 帮助用户把被动词汇转化为主动词汇，提升实际表达中的用词准确度。
2. 通过选择题练习降低表达练习的启动门槛，用户无需事先储备好词即可开始。
3. 通过填空题训练用户主动调用词汇的能力。
4. 通过写作评分提供最高难度的综合表达练习，给出具体、可操作的用词改进建议，而非泛泛评价。
5. 支持用户围绕自己感兴趣的话题定制练习内容，保持练习的针对性和持续性。
6. 本地运行，业务数据本地保存；无需 fitword 账号，模型访问依赖用户可用的 LLM 服务。

## 非目标

v0.1 明确不做：

1. 社交功能（排行榜、分享、社区）
2. 完整课程体系或学习路径
3. 移动端
4. 复杂账号系统
5. 大规模内置题库
6. URL 抓取（v0.1 仅支持用户粘贴文本）
7. 语音输入
8. 浏览器插件
9. 词语收藏夹

## 范围

v0.1 包含：

1. 选择题与填空题练习：话题选择 → LLM 出题 → 用户作答 → 反馈讲解 → 错题记录
2. 写作评分：用户输入文字 → 五维评分 → 可替换词建议 → 改写版本 → 本地保存评分记录 → 衍生选择题
3. 话题定制：用户粘贴文本 → LLM 生成对应领域的练习素材
4. 答题记录、错题记录与写作评分记录：本地持久化
5. 本地 Web UI 交互界面

## 成功标准

1. 用户可以在 1 分钟内开始第一道练习。
2. 用户不用自己先想出好词，也能完成练习。
3. 每次练习后，用户至少获得 1 个可复用词语或表达。
4. 写作评分能给出具体可改的建议，而不是泛泛评价。

## 后续方向（v0.1 之后）

1. 每日 10 题练习
2. 错题复习
3. 词语收藏夹
4. 语音输入，练习口语表达
5. 浏览器插件：选中网页内容后生成练习
6. 更精细地根据用户错题自动调整难度和题型
