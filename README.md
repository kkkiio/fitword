# fitword（词感）

一个本地运行的 LLM 表达练习工具，通过选择题和填空题帮你把"被动词汇"转化为"主动词汇"。核心思路是让 LLM 提供候选词、语境和反馈，把表达练习从"自己硬想"降到"辨认、选择、产出"，让你不需要事先想好词，也能完成有效练习。

## 使用

启动后，fitword 以 chatbot 对话形式与你交互。Agent 会在对话中主动出题——选择题（四选一）或填空题（自己打出答案），你作答后 Agent 给出反馈。

```
用户: 最近写周报总觉得用词太干了，想练练
Agent: 写周报确实容易落入套路。你在周报里通常描述哪类内容？
用户: 主要是进度汇报
Agent: 那来一道动词选择题——

  项目第一阶段的开发已经____，下周进入测试。
  A. 完成  B. 告一段落  C. 收尾  D. 结束

用户: B
Agent: 对，B 最好。比"完成"多了"阶段性停顿、后续还有"的含义。
```

三种练习类型：

- **选择题**：4 个候选词中选出最贴切的
- **填空题**：自己打出答案，Agent 评价并提供更好的词
- **写作评分**：贴入一段文字，获得五维评分和改写建议

所有数据存储在本地 `~/.fitword/`，无需网络、无需账号。

## 功能

- 5 种知识类别：名词、动词、形容词、逻辑词、领域词
- 选择题 + 填空题双格式
- 五维写作评分（准确度、具体度、自然度、结构、语域）
- Agent Memory：Agent 根据你的练习历史调整出题策略
- 完全本地运行，数据由你掌控
